## ⚡ Sistema de Combate

| Feature | Descrição |
|---------|-----------|
| **AimLock** | Mira automática no alvo mais próximo com FOV ajustável |
| **Trigger Bot** | Dispara automaticamente ao mirar em um inimigo |
| **Wall Check** | Só mira se não tiver parede no meio |

---

## 🧠 Movimento

| Feature | Descrição |
|---------|-----------|
| **Fly** | Speed Fly ajustavel |
| **Noclip** | Atravesse qualquer objeto |
| **Speed** | Velocidade ajustável de 1 a 200 |
| **Inf Jump** | Pulos infinitos sem parar |

---

## 👁️ Visual

| Feature | Descrição |
